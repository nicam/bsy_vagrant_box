bsy_vagrant_box
===============

# Vagrant Setup
http://docs.vagrantup.com/v1/docs/getting-started/index.html

Run the vagrant box

	vagrant up

Open ddd debug with X11 forwarding

	sh open_ddd.sh

Connect to the box

	vagrant ssh
	
For mountain lion users:

you need to install x11 to use x11 forwarding
http://xquartz.macosforge.org/landing/
